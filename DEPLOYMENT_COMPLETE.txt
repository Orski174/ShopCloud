KEY COMMANDS
============

View deployment status:
  aws ecs describe-clusters --clusters shopcloud-dev --region us-east-1

Check ECS tasks:
  aws ecs list-tasks --cluster shopcloud-dev --region us-east-1

View CloudWatch logs:
  aws logs tail /ecs/shopcloud-dev/auth --follow --region us-east-1

Get ALB DNS name:
  aws elbv2 describe-load-balancers --query "LoadBalancers[?LoadBalancerName=='shopcloud-public-dev'].DNSName" --region us-east-1

List ECR repositories:
  aws ecr describe-repositories --query "repositories[].repositoryName" --region us-east-1
